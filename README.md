# PersonalBudgetTracker

## Projektbeskrivning:

PersonalBudgerTracker är en konsolapplikation där man hanterar sin ekonomi. Du kan registrera inkomster och utgifter, se tidigare transaktioner, ta bort transaktioner och se din aktuella balans.

## Hur man kör programmet:

1. Klona ner repot till din dator.
2. Öppna projektet i din favorit IDE (ex. Visual Studio, VS Code).
3. I VS Code kan du trycka på knappen med den gröna liggande bilen i mitten, längst upp på din skärm. Men du kan också i välja att start igenom att öppna terminalen och köra "dotnet run". (I mappen för projektet)
4. Följ bara instruktionerna i konsolen, du kommer få flera val att välja emellan och applikationen kommer fortsätta tills du väljer att stänga den.
